# Social Media Client

> Workflow CA for Ingrid Skeide

## Requirements
- NodeJS
- NPM

## Installation
1. Clone the repository
```
git clone <repository-url>
cd social-media-client
```

2. Install dependencies
```
npm install 
```

3. Initialize Husky and Git Hooks
```
npx husky install
```

## Commands
1. Start development server
```
npm run start
```

2. Build CSS
```
npm run build
```

3. Format, lint, test
```
npm run format
npm run lint
npm run test
```
